# CoffeeShop

## Описание
**CoffeeShop** — это корпоративное приложение для управления кофейней, разработанное с использованием технологии WPF на языке программирования C#. Приложение предназначено для управления различными аспектами деятельности кофейни, включая управление запасами, обработку заказов, управление сотрудниками и генерацию отчетов.

## Технологии и Инструменты
- **C# и .NET:** Основной язык программирования и фреймворк.
- **WPF (Windows Presentation Foundation):** Создание интерактивных пользовательских интерфейсов.
- **Entity Framework:** ORM для взаимодействия с базой данных.
- **Microsoft SQL Server:** Система управления базами данных.
- **DocX:** Библиотека для создания и редактирования документов Word.

## Функционал

### Управление запасами
- Просмотр текущих остатков на складе.
- Добавление новых продуктов и категорий.
- Принятие поставок и обновление запасов.
- Фильтрация и поиск по складу.

### Обработка заказов
- Создание и обработка заказов.
- Поиск и просмотр чеков.
- Генерация отчетов по заказам.

### Управление сотрудниками
- Создание, редактирование и удаление сотрудников.
- Назначение пин-кодов для сотрудников.
- Просмотр личных данных и статистики по сотрудникам.

### Отчеты и аналитика
- Генерация отчетов по поставкам, продажам и сотрудникам.
- Генерация отчетов по остаткам на складе.
- Печать отчетов и сохранение в формате Word.

## Основные компоненты

### Главная страница администратора (AdminMainPage)
- Управление поставщиками.
- Управление продуктами.
- Управление сотрудниками.
- Просмотр и управление кассовыми сменами.

### Управление продуктами (ManageProduct)
- Просмотр, добавление, обновление и удаление продуктов.
- Создание и управление категориями продуктов.

### Управление сотрудниками (ManageStaffPage)
- Просмотр, добавление, обновление и удаление сотрудников.
- Назначение и управление пин-кодами сотрудников.

### Просмотр остатков на складе (ViewStockPage)
- Просмотр текущих запасов на складе.
- Фильтрация и поиск по складу.

### Обработка заказов и чеков (CashierWindowForm)
- Создание новых заказов.
- Просмотр и поиск по чекам.
- Генерация и печать отчетов по заказам.
